To use this project, you need to move the TURT.LISP file into your Medley environment. Then either load it from the file browser OR LOAD it from your Interlisp REPL (Exec).
Do not attempt to paste the file directly.
You can view the included manual TURT.TEDIT in TEDIT.
You can of course, compile the package to make it much faster. :)
